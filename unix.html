I'm not sure why but messing around with operating systems, particularly UNIX, is a big hobby of mine. I even enjoy reading textbooks on operating systems design (for fun!) and have read (and worked through the source) The Design and Implementation of the FreeBSD Operating System.

"I think vim and sed tie in the "coolest program ever" debate. There really isn't any argument...unless you count pipes."

What I currently use

  • Apple OS X (latest) - my main os
  • FreeBSD - my favorite.
  • Windows - gaming system

Believe it or not but I get asked quite a bit about which types of UNIX and Linux I've used. Well, here is the official list: RedHat, Fedora Core, SuSe, Debian, Slackware, Gentoo, OpenBSD, FreeBSD, Sun Solaris, Open Solaris, Apple OS X, Ubuntu

Of the above, I really only use Ubuntu on a VM and OS X anymore because my needs have changed. I know some people slam Ubuntu but hey I used Debian ten years ago and I liked it. In the many years I have been using Linux and UNIX (from 2001 onward) these operating systems have come a long way and the future looks good.

I've enjoyed using OS X since it was first released and currently it is the only os I use in any real capacity.

In 2006 I started using subversion to store my life. Sometime in 2012 I moved my life to git. I'd say give it a try, having your $HOME under change control is a very good thing.
